# Приложение для поиска шаблонов успеваемости студентов на основе данных портала Универис
## Елисеев Е.В.

Проект разрабатывает приложение для анализа данных об успеваемости студентов, хранящихся в системе Универис. Приложение использует метод ассоциативных правил для выявления сильных связей и закономерностей между элементами в больших наборах данных. Это позволяет пользователям проводить анализ на выборке, которая им нужна, и находить в ней закономерности, на основе которых можно сделать выводы.

В проекте используется фреймворк Dash для создания пользовательского интерфейса.

## Установка

Установите Python 3.11.7

1. Клонируйте репозиторий на ваш локальный компьютер.
2. Установите необходимые зависимости, указанные в файле `requirements.txt`. Вы можете сделать это, используя команду `pip install -r requirements.txt`.
3. Запустите приложение, выполнив файл `univeris_pattern_miner_app.py`.
